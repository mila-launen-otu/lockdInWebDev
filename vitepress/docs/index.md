    title: Real-Time Chat
    details: Message classmates instantly with live socket-powered chat. No refresh needed — conversations flow in real time, keeping your study session connected and seamless.

  - icon: 🎨
    title: Collaborative Whiteboard
    details: Draw, annotate, and brainstorm together on a shared canvas. Choose colors, brush sizes, and erase freely. Every stroke syncs live across all connected users.

  - icon: ❓
    title: Questions & Answers
    details: Post homework questions and field answers from the community. Submissions persist via local storage so your Q&A history is always there when you need it.

  - icon: 📊
    title: Grades Tracker
    details: Keep track of your academic performance in one place. Log assignments with scores and weights per class. Get a weighted grade percentage, letter grade, and GPA points automatically. Saves to CSV on the backend and loads back on login.

  - icon: ⚡
    title: Socket.IO Powered
    details: Built on Socket.IO for low-latency, bidirectional communication. Every feature that needs real-time sync — chat, whiteboard, Q&A — runs on the same robust backend.

  - icon: 🔧
    title: Vue 3 + Vite Frontend
    details: Snappy, component-based SPA architecture using Vue 3 with the Composition API and Vue Router, served at blazing speed by Vite's dev server.
